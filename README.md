# Microprocessor-with-Arduino-Interface

## ISA Summary
**Page 1 (Base 16):** LDI, ADD, SUB, ADI, MUL, DIV, DEC, INC, NOR, NAND, XOR, COMP, CMPJ, JMP, HLT

**Page 2 (Extended via PREFIX):** PUSH, POP, CALL, RET, LDM, STM, SHL, SHR, AND, OR, MMRD, MMWR, ROTL, ROTR, SUBI, RETI

## Tools
- SystemVerilog
- Quartus Prime Lite 20.1
- ModelSim Intel FPGA 2020.1
- Arduino IDE (SPI master sketch)

## Target Hardware
- DE10-Lite (Intel MAX10 10M50DAF484C7G)
- Arduino Uno (SPI master)
